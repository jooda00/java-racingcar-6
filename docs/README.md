### 기능
1. 자동차 입력 받기
* 이름 구분은 쉼표
* 예외기준
    
    이름은 5자 이하
  
    동일한 이름은 불가함

    자동차 이름이 공백이거나 null인 경우

    자동차가 한 대인 경우
  
2. 시도할 횟수 입력 받기
* 횟수는 0보다 큰 수
* 예외기준

    0보다 같거나 작은 수
  
3. 게임 실행
* 0~9사이의 무작위 수를 구해서 4이상인 경우 전진
* 각 사용자마다 무작위 수 구하기
* 예외기준

    
4. 우승자 정하기
* 우승자는 1명 이상일 수 있음
* 전진횟수로 내림차순. 동점이 나오면 입력한 사용자 순서대로 쉼표로 구분하여 출력
* 예외기준

    아무도 전진한 사람이 없을 때
